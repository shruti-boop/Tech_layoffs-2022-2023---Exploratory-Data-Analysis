# Tech Company Layoffs – Exploratory Data Analysis

## * Project Overview
This project performs Exploratory Data Analysis (EDA) on technology company layoffs to identify trends across companies, industries, locations, and time periods.

## * Dataset
- Dataset: Technology Company Layoffs Dataset
- Columns include: company, total_layoffs, industry, location, reported_date, status, etc.
- Data cleaning and preprocessing were carried out prior to analysis.

## * Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Tableau

## * Key Analysis Performed
- Data loading and initial inspection

- Handling missing values and duplicates

- Univariate and bivariate analysis

- Company-wise, industry-wise, and location-wise layoff analysis

- Time-based trends (2022–2023)

- Workforce impact analysis

## * Files in Repository
- Jupyter Notebook for EDA
- PowerPoint presentation summarizing insights
- Tableau Dashboard for interactive visualization
- Original dataset used for analysis

## * Conclusion
The analysis highlights major layoff trends in the tech industry and identifies industries and regions most affected.
